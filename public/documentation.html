myFlix Movies

What the API does:

  • Returns a list of all movies to the user.
  • Returns data (description, genre, director, image URL, and whether it's featured or not) about a single movie by title to the user.
  • Returns data about a genre (description) by name/title.
  • Returns data about a director (bio, birth and death year) by name.
  • Allows new users to register.
  • Allows users to update their user info (username, password, email, and date of birth).
  • Allows users to add/remove a movie to their list of favorites.
  • Allows existing users to deregister.
